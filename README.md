# Commodity and Currency Market Analysis

This project analyses Brent Oil, Gold, Silver and Swift Currency Tracker datasets.

## Questions Completed

- A5: Main Metrics Available in the Swift Currency Tracker Dataset
- B9: Average Price by Year for Brent Oil, Gold and Silver
- B12: Five Years with the Highest Average Silver Prices
- C16: Combined Chart Showing Gold and Silver Trends
- E27: Commodity with the Largest Visible Price Swings

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Kaggle Notebook
